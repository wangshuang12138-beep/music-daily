# 🎵 xca's pick | 每日音乐推荐

我的个人音乐推荐站，每天分享一首歌。

## 🌐 在线访问

**https://wangshuang12138-beep.github.io/music-daily/**

## ✨ 功能

- 📅 **每日推荐** - 以日期为维度，每天一首精选音乐
- 🏷 **标签筛选** - 支持按标签（晨间/电子/华语等）筛选历史推荐
- 🎧 **沉浸播放** - 内嵌网易云播放器，支持外链跳转
- 📱 **移动端适配** - 完美支持手机端浏览
- 🎨 **沉浸式视觉** - 全屏专辑封面背景 + 毛玻璃效果

## 📝 如何添加每日推荐

编辑 `data.json` 文件，按日期格式添加：

```json
{
  "2026-03-16": {
    "title": "歌曲名",
    "artist": "歌手",
    "album": "专辑名",
    "cover": "专辑封面图URL",
    "tags": ["晨间", "轻快", "华语"],
    "neteaseId": "网易云歌曲ID",
    "description": "推荐文案...",
    "quote": "一句歌词金句"
  }
}
```
